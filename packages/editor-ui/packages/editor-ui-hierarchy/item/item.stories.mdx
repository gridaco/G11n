## Select Item & UnSelect Item

**Select Item** : Background color changes when users perform events such as click.

**UnSelect Item** : Changes to Select Item when an unselected item, mouse-hover event, or click event occurs

## Item Level Variant (0 ~ 3)

**Struct's Deps** are called **levels**

Margin formula for level :

```js
const levelMargin = 14;
const defaultMargin = 25;

return level * levelMargin + defaultMargin;
```
